# (Uibot) excel IO&manage script 台账进出管理脚本
基于Uibot creator 5.1.1开发的Excel台账管理脚本，包含功能：

1.将分散的表格和文件批量导入台账，例如分发客户或下属机构填报的表单或大批量业务表单，一键批量导入到台账（内容读取和转写可自定）；

2.台账选取批量行分别导出为单个文件，例如由总表管理的客户信息在需要时导出为业务表单用以分发（导出格式可自定）；

3.*开发中* 在台账规定位置捕获到关键字后按规则批量转写，例如按照台账登记的公司全称批量备注对应的行业类型（抓取和转写规则可自定）；

4.*开发中* 提取台账信息关联文件的文件名或内容并将匹配情况反馈到台账，例如比对台账和档案文件进行查漏（匹配方式可自定，目前读取内容仅支持文档/表格）。


*未来也可能继续开发其他有关功能，加入OCR，实现票据等图片数据批量导入或信息匹配。

主要功能所需的参数均可由excel表格进行配置管理，可自定义需要管理的台账、处理的文件、读取写入导出等各种规则。
